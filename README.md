# Haoyang Guo — Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my background, interests, and projects.  
Designed and developed as part of **IDSN 531: Front-End Web Programming** at the University of Southern California (USC Iovine and Young Academy).

---

## Overview

This website serves as both a creative introduction and a technical showcase of my front-end development skills.  
It features multiple sections, a consistent dark theme, and elegant typography inspired by cinematic minimalism.

**Key features include:**
- **Welcome Page** – A clean landing page with a hero section introducing who I am.  
- **Bio Page** – A “Z-layout” story section describing my journey from computer science to design and technology.  
- **Projects Page** – A responsive project grid highlighting selected works and experiments.  
- **Gallery Page** – A horizontally scrollable photo section featuring my cat “Kiki” and travel memories.  
- **Contact Page** – A styled contact form with real-time input validation using jQuery.  
- **Theme & Typography** – Elegant serif titles and smooth transitions across all pages.  
- **Responsive Design** – Fully optimized for desktop, tablet, and mobile screens.
